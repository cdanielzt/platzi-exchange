# Platzi exchange

Visualiza las principales criptomonedas, historial, precio actual y mejores ofertas.

Proyecto realizado para el curso básico de Vue.js 2 de Platzi utilizando la [API de Coincap](https://docs.coincap.io/).


## Tecnologías usadas
- Vue Js
- Tailwind
- Apí fetch
